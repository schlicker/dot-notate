# dot-notate

dot-nonte converts an object into a one dimensional object with its attributes separated by '.'.

## Example

```
var dotNotate = require('dot-notate');

console.log(
    dotNotate({
        this: {
            is: 'an example';
        }
    })
);
```

## Installation

```
npm install --save dot-notate
```
